# 🌐 Overview

**GNP English Center Website** là website giới thiệu trung tâm Anh ngữ với thiết kế **hiện đại, nhiều tương tác và thân thiện với người dùng**.

Website được xây dựng theo phong cách **EdTech hiện đại**, lấy cảm hứng từ các trung tâm tiếng Anh quốc tế như **Apollo English**.

Màu sắc chủ đạo của website là **cam năng động**, giúp tạo cảm giác tích cực và truyền động lực học tập.
